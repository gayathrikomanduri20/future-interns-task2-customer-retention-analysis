## Customer Retention & Churn Analysis

## Objective
Analyze customer churn behavior and identify key retention drivers in a subscription-based telecom company.

## Business Problem
Why are customers leaving?
Which segments churn the most?
How long do customers stay active?
What actions can reduce churn?

## Dataset
Telco Customer Churn Dataset (7,044 customers)

## Tools Used
- Microsoft Excel
- Data Cleaning
- Feature Engineering
- Pivot Tables
- Pivot Charts
- Dashboard Design

## Key Metrics
- Total Customers: 7044
- Churned Customers: 1869
- Churn Rate: 26.54%
- Retention Rate: 73.46%
- Average Customer Lifetime: 32.37 months

## Key Insights
1. Month-to-month contracts show highest churn (42.71%).
2. Customers within first 3 months show extreme churn (56.21%).
3. Electronic check users have highest churn (45.29%).
4. Long-term contracts significantly reduce churn (2.83%).

## Business Recommendations
- Improve onboarding during first 90 days.
- Promote annual and long-term plans.
- Investigate electronic check payment issues.
- Launch early retention campaigns.

## Conclusion
Early lifecycle management and contract commitment are the strongest retention drivers.
